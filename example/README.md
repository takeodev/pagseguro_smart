# PagSeguro Smart - Exemplo

Exemplo de integração com o pacote `pagseguro_smart`.

## Funcionalidades
- Ativação de PinPad
- Pagamento com Cartão de Crédito/Débito
- Pagamento via PIX e Voucher
- Estorno e Cancelamento
- Visualização do dono da maquininha

## Como executar
1. Conecte a maquininha Smart 2.
2. Altere o código de ativação no campo inicial.
3. Execute o app em modo release (`flutter run --release`).
